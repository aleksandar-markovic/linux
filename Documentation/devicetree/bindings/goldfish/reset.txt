Android Goldfish reset

Android goldfish reset device generated by android emulator.

Required properties:

- compatible : should contain "google,goldfishi-reset" to match emulator
- reg        : <registers mapping>
- interrupts : <interrupt mapping>

Example:

	goldfish_reset@9020000 {
		compatible = "google,goldfish-reset";
		reg = <0x9020000 0x1000>;
		interrupts = <0x3>;
	};
