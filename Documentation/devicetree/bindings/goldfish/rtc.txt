Android Goldfish RTC

Android goldfish RTC device generated by android emulator.

Required properties:

- compatible : should contain "google,goldfish-rtc" to match emulator
- reg        : <registers mapping>
- interrupts : <interrupt mapping>

Example:

	goldfish_rtc@9020000 {
		compatible = "google,goldfish-rtc";
		reg = <0x9020000 0x1000>;
		interrupts = <0x3>;
	};
