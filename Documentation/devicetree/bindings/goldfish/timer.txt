Android Goldfish timer

Android goldfish timer device generated by android emulator.

Required properties:

- compatible : should contain "google,goldfish-timer" to match emulator
- reg        : <registers mapping>
- interrupts : <interrupt mapping>

Example:

	goldfish_timer@9020000 {
		compatible = "google,goldfish-timer";
		reg = <0x9020000 0x1000>;
		interrupts = <0x3>;
	};
