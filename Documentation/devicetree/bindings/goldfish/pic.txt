Android Goldfish PIC

Android goldfish PIC device generated by android emulator.

Required properties:

- compatible : should contain "google,goldfish-pic" to match emulator
- reg        : <registers mapping>
- interrupts : <interrupt mapping>

Example:

	goldfish_pic@9020000 {
		compatible = "google,goldfish-pic";
		reg = <0x9020000 0x1000>;
		interrupts = <0x3>;
	};
